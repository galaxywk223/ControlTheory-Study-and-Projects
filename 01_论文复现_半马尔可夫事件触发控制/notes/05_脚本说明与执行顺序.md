# 脚本说明与执行顺序

## 1. 脚本整理结果

目前脚本已经按用途分成两组：

### 1.1 Python 原型

路径：

- `scripts/python/dc_motor_mdsetm_simulation.py`

作用：

- 展示 DC 电机算例的整体流程；
- 生成切换信号、状态响应、控制输入和事件间隔图；
- 适合作为阅读入口。

当前状态：

- 已整理；
- 文件名已统一；
- 可以作为公开展示脚本。

### 1.2 MATLAB 分阶段主线

路径：

- `scripts/matlab/zie_pipeline/`

包含脚本：

1. `zIE_01_setup_paths_and_system.m`
2. `zIE_02_stage0_K.m`
3. `zIE_03_stage1_observer.m`
4. `zIE_04_stageB_full_trigger.m`
5. `zIE_05_sim_min_ET_nohatw.m`
6. `zIE_06_sim_ET_hatw.m`
7. `zIE_07_sweep_sigma.m`

作用：

- 定义系统与参数；
- 求解控制器和观测器相关量；
- 构造触发矩阵；
- 进行事件触发仿真；
- 扫描参数并观察折衷关系。

当前状态：

- 已按阶段归档；
- 已修正脚本内部读写路径；
- 当前保留原始命名风格，便于与已有脚本一一对应；如需展示可再统一为更直观的名称。

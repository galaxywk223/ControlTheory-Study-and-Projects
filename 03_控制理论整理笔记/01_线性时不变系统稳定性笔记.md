# 线性时不变系统稳定性笔记

## 线性时不变系统概述

线性时不变系统（Linear Time-Invariant System, LTI）同时满足：

- 线性；
- 时不变性；
- 系统映射关系明确。

可以简单理解为：输入和输出之间的关系既满足线性叠加规律，又不会随时间改变。

## 系统、线性与时不变性

### 系统

系统可以看作一个“黑箱”：给定输入，就会产生相应输出。

### 线性

线性包含两个基本性质：

1. 叠加性：若输入 $x_1,x_2$ 分别产生输出 $y_1,y_2$，则输入 $x_1+x_2$ 会产生输出 $y_1+y_2$。
2. 齐次性：若输入 $x$ 产生输出 $y$，则输入 $a x$ 会产生输出 $a y$，其中 $a$ 为任意常数。

### 时不变性

时不变性表示：若输入信号只是发生时间平移，那么输出也只会相应平移，而不会改变系统本身的映射规律。

对于系统

```math
\dot x(t)=Ax(t),
```

若 $x_1(t),x_2(t)$ 都是系统解，则

```math
c_1x_1(t)+c_2x_2(t)
```

仍然是系统解，这体现了线性；若 $x(t)$ 是系统解，则 $x(t+s)$ 仍满足同样的演化形式，这体现了时不变性。

## 系统稳定性

考虑自治动力系统

```math
\dot x = f(x(t)), \qquad x(t_0)=x_0.
```

若存在 $a$ 使得 $f(a)=0$，则称 $a$ 为系统平衡点。

### 数学背景简述

为了讨论稳定性，常用到赋范向量空间与完备性：

- 向量空间：满足加法与数乘封闭。
- 赋范向量空间：在向量空间上定义范数，用来度量向量长度。
- 完备赋范向量空间：所有柯西序列都在空间内收敛，也称 Banach 空间。

## 李雅普诺夫稳定

若对任意 $\varepsilon > 0$，都存在 $\delta(\varepsilon)>0$，使得

```math
\|x_0-a\|<\delta \quad \Longrightarrow \quad \|x(t)-a\|<\varepsilon,\ \forall t\ge t_0,
```

则称平衡点 $a$ 是李雅普诺夫稳定的。

直观上，这意味着：如果初始状态足够靠近平衡点，那么系统轨迹在未来始终不会偏离太远。

### 说明

平衡不一定意味着稳定。

- 碗底的小球：轻微扰动后仍在附近运动，对应稳定。
- 倒扣碗顶的小球：轻微扰动后会滚落，对应不稳定。

## 渐近稳定

在李雅普诺夫稳定基础上，若还满足

```math
\lim_{t\to\infty}x(t)=a,
```

则称平衡点 $a$ 是渐近稳定的。

直观上，渐近稳定不仅要求“不会跑远”，还要求“最终收敛回平衡点”。

## 指数稳定

若在渐近稳定基础上，还存在常数 $\alpha,\beta,\delta>0$，使得

```math
\|x(t)-a\| \le \alpha \|x_0-a\| e^{-\beta t},
```

则称平衡点 $a$ 指数稳定。

这说明系统状态不仅收敛，而且收敛速度至少和指数函数一样快。

## 渐近稳定判据

考虑连续时间 LTI 系统

```math
\dot x(t)=Ax(t), \qquad x(0)=x_0.
```

其中原点 $x=0$ 是平衡点。

### 特征值判据

对于连续时间线性时不变系统，原点渐近稳定的充分必要条件是：

```math
A \text{ 的全部特征值实部都为负。}
```

其直观含义是：系统各方向上的模态都会随时间衰减，因此状态最终会回到原点。

### 李雅普诺夫判据

对于 $n$ 维连续时间线性时不变系统，原点渐近稳定的充分必要条件是：

存在对称正定矩阵 $P$，使得

```math
A^T P + P A \prec 0.
```

这就是经典的连续时间李雅普诺夫判据。

## 指数稳定的证明思路

考虑系统

```math
\dot x(t)=Ax(t),
```

假设存在矩阵 $P=P^T\succ 0$，使得

```math
A^T P + P A \prec 0.
```

则系统指数稳定。

### 证明主线

构造李雅普诺夫函数

```math
V(t)=x^T(t)Px(t). \tag{1}
```

由于 $P\succ 0$，所以 $V(t)>0$。

对式 (1) 求导可得

```math
\dot V(t)=x^T(t)(A^TP+PA)x(t). \tag{2}
```

再利用牛顿-莱布尼兹公式，可得到

```math
e^{kt}V(t)=V(0)+\int_0^t e^{k\theta}\big[kV(\theta)+\dot V(\theta)\big]\,d\theta. \tag{3}
```

另一方面，由矩阵 $P$ 的特征值性质可得

```math
\lambda_{\min}(P)\|x(t)\|^2 \le V(t) \le \lambda_{\max}(P)\|x(t)\|^2. \tag{4}
```

将 (3) 与 (4) 联立，并选择足够小的 $k>0$，使得相关积分项系数为负，就能推出

```math
\|x(t)\|^2 \le \vartheta e^{-kt},
```

从而得到指数稳定结论。

## 求解与仿真

考虑如下系统：

```math
x(t)=
\begin{bmatrix}
x_1(t) \\
x_2(t)
\end{bmatrix},
\qquad
A=
\begin{bmatrix}
0 & 1 \\
-1 & -1
\end{bmatrix}.
```

### MATLAB 求解思路

原始文档给出了一段基于 MATLAB LMI 工具箱的求解代码，其核心流程如下：

1. 定义系统矩阵 $A$。
2. 初始化 LMI 系统 `setlmis([])`。
3. 用 `lmivar` 定义对称矩阵变量 $P$。
4. 用 `lmiterm` 构造李雅普诺夫不等式

```math
A^T P + P A \prec 0.
```

5. 用 `feasp` 求解可行解。
6. 用 `dec2mat` 还原矩阵 $P$。

### 关键命令说明

- `setlmis([])`：初始化一个新的 LMI 问题。
- `lmivar(1,[2,1])`：定义一个 $2\times2$ 满对称矩阵变量。
- `lmiterm([1,1,1,P],1,A,'s')`：向第 1 个 LMI 的 $(1,1)$ 块添加对称项 $A^TP+PA$。
- `feasp(lmis)`：求解该 LMI 是否有可行解。

### 说明

这一部分代码解释的核心目的是说明：

- 如何把稳定性条件转化为 LMI；
- 如何在 MATLAB 中定义未知矩阵变量；
- 如何利用工具箱自动求解李雅普诺夫矩阵 $P$。

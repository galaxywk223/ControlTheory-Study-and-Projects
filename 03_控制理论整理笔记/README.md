# 控制理论整理笔记

本目录整理了一组控制理论学习笔记，内容从线性时不变系统的基础稳定性开始，逐步扩展到控制综合、采样控制、鲁棒控制，再到非线性时滞神经网络和混沌同步问题。

## 阅读顺序

建议按以下顺序阅读，整体难度由简单到复杂：

1. `01_线性时不变系统稳定性笔记.md`
2. `02_线性时不变系统控制笔记.md`
3. `03_线性时不变系统周期采样控制与稳定性分析笔记.md`
4. `04_鲁棒控制笔记.md`
5. `05_非线性时滞神经网络稳定性笔记.md`
6. `06_混沌时滞神经网络同步与图像加密笔记.md`

## 内容说明

- `01_线性时不变系统稳定性笔记.md`
  - 介绍线性时不变系统、平衡点、李雅普诺夫稳定、渐近稳定和指数稳定等基础概念。
- `02_线性时不变系统控制笔记.md`
  - 记录线性时不变系统的状态反馈、输出反馈以及相关 `H_\infty` 控制条件。
- `03_线性时不变系统周期采样控制与稳定性分析笔记.md`
  - 讨论周期采样状态反馈、时滞建模、Lyapunov-Krasovskii 泛函和稳定性分析。
- `04_鲁棒控制笔记.md`
  - 整理鲁棒控制中的扰动抑制、`\mathcal{H}_\infty`、`\mathcal{L}_2-\mathcal{L}_\infty` 等基本内容。
- `05_非线性时滞神经网络稳定性笔记.md`
  - 进入非线性系统和时滞神经网络稳定性分析。
- `06_混沌时滞神经网络同步与图像加密笔记.md`
  - 包含混沌系统同步、误差系统分析以及图像加密应用，综合性最强。

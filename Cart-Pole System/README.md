# Cart-Pole System

本目录记录倒立摆系统的三条控制研究路线：

- 经典控制：基于物理模型建立线性状态空间模型并设计状态反馈控制器；
- 系统辨识：通过输入输出数据辨识模型，再进行验证；
- 强化学习：构造环境并直接学习控制策略。

## 当前进度

- 已完成：倒立摆线性模型整理与闭环仿真。
- 已完成：基于仿真数据的数据采集、参数辨识与模型验证。
- 已完成：强化学习环境搭建、PPO 训练脚本和结果分析脚本整理。
- 已完成：脚本、图片、笔记的目录重构。
- 待完成：三条路线的统一结果对照与指标汇总。

## 内容说明

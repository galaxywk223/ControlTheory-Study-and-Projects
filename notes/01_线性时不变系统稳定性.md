# 线性时不变系统稳定性

这篇笔记整理连续时间线性时不变系统的稳定性判定。核心问题很直接：给定系统

```math
\dot x(t)=Ax(t),
```

如何判断平衡点是否稳定、渐近稳定，以及是否具有指数收敛速度。本文主要保留两条最常用的判定思路：特征值判据和李雅普诺夫判据，并用一个二维系统做数值验证。对应实验见 [`scripts/foundations/01_lti_stability`](../scripts/foundations/01_lti_stability/README.md)。

## 基本概念

对自治系统

```math
\dot x=f(x), \qquad x(t_0)=x_0,
```

若存在平衡点 $a$ 使得 $f(a)=0$，则有：

- 李雅普诺夫稳定：初值足够靠近 $a$ 时，轨迹始终留在其邻域内。
- 渐近稳定：在李雅普诺夫稳定基础上，还满足 $\lim_{t\to\infty}x(t)=a$。
- 指数稳定：不仅收敛，而且满足指数衰减上界。

```math
\|x(t)-a\| \le \alpha \|x_0-a\|e^{-\beta t}.
```

## 两个常用判据

对于连续时间 LTI 系统

```math
\dot x(t)=Ax(t),
```

原点渐近稳定当且仅当矩阵 $A$ 的全部特征值实部都小于零。这是最直接的谱判据。

另一条等价结论是李雅普诺夫判据：若存在对称正定矩阵 $P$，使得

```math
A^TP+PA\prec 0,
```

则原点渐近稳定。若进一步能建立

```math
\lambda_{\min}(P)\|x(t)\|^2 \le V(t)=x^T(t)Px(t)\le \lambda_{\max}(P)\|x(t)\|^2
```

并证明 $\dot V(t)\le -kV(t)$，则可推出指数稳定。

## 证明主线

李雅普诺夫法的关键不在于把证明展开得很长，而在于抓住两个量：

1. 选一个正定函数 $V(t)=x^T(t)Px(t)$。
2. 证明它沿轨迹严格下降，即

```math
\dot V(t)=x^T(t)(A^TP+PA)x(t)<0.
```

一旦 $P\succ0$ 且 $A^TP+PA\prec0$，状态范数就能被 $V$ 的上下界夹住，从而得到收敛性和收敛速度估计。

## 数值验证

考虑二维系统

```math
A=
\begin{bmatrix}
0 & 1 \\
-1 & -1
\end{bmatrix},
\qquad
x(0)=
\begin{bmatrix}
1 \\
-1
\end{bmatrix}.
```

其特征值为

```math
\lambda_{1,2}=-\frac{1}{2}\pm\frac{\sqrt{3}}{2}i,
```

实部均为负，因此系统渐近稳定。对李雅普诺夫方程

```math
A^TP+PA=-I
```

求得一组正定解

```math
P=
\begin{bmatrix}
1.5 & 0.5 \\
0.5 & 1
\end{bmatrix},
```

说明谱判据和李雅普诺夫判据在这个例子上给出一致结论。

## 小结

对连续时间 LTI 系统，稳定性分析最常用的入口仍然是特征值位置和李雅普诺夫方程。前者适合快速判断，后者适合继续推导鲁棒性、性能分析和 LMI 条件。这个二维示例同时满足两类判据，数值结果也与理论分析一致。

# 线性时不变系统控制

这篇笔记整理 LTI 系统中的状态反馈与输出反馈设计。笔记重点说明如何把“稳定闭环并满足性能指标”转成可求解的 LMI 条件，并通过变量代换恢复控制器增益。对应实验见 [`experiments/foundations/02_lti_control`](../experiments/foundations/02_lti_control/README.md)。

## 问题设置

考虑系统

```math
\dot x(t)=Ax(t)+Bu(t)+D\omega(t), \qquad y(t)=Cx(t).
```

控制律分为两类：

- 状态反馈：$u(t)=Kx(t)$
- 输出反馈：$u(t)=Ky(t)=KCx(t)$

目标是在闭环稳定的同时满足给定的 $H_\infty$ 性能指标，即让扰动 $\omega$ 到输出 $y$ 的 $L_2$ 增益小于某个常数 $\gamma$。

## 分析与综合的基本思路

若控制器已经给定，可以先做分析：寻找 $P\succ0$ 使

```math
\dot V(x)+y^Ty-\gamma^2\omega^T\omega<0,
\qquad
V(x)=x^TPx.
```

这样就能把稳定性和性能要求写成矩阵不等式。

若控制器还未知，则需要做综合。困难在于 LMI 中会出现 $PK$、$KX$ 这类双线性项。常见做法是引入新变量，例如

```math
L=KX \quad \text{或} \quad L=PK,
```

把非凸条件改写成关于 $(X,L)$ 或 $(P,L)$ 的线性矩阵不等式，最后再由

```math
K=LX^{-1}
\quad \text{或} \quad
K=P^{-1}L
```

恢复反馈增益。

## 状态反馈与输出反馈

状态反馈设计的结构最直接，闭环项通常以

```math
\mathrm{He}(AX+BL)
```

的形式出现，因此更容易得到标准 LMI。

输出反馈设计更复杂，因为闭环矩阵中含有

```math
A+BKC,
```

对应项 $PBKC$ 同时耦合 $P$ 和 $K$。因此通常需要更多辅助变量，或者采用两阶段设计与参数恢复方法。不同定理在形式上不同，但核心都是把不可直接求解的双线性问题转成可行性 LMI。

- 不同 LMI 设计对应的瞬态形状不同，有的振荡更明显，有的更平滑；
- 同样是稳定化控制，设计条件的松弛方式会直接影响闭环响应特征。

## 小结

线性系统控制设计的核心工作，通常就是把闭环稳定和性能约束写成可求解的矩阵不等式。状态反馈结构更便于基准比较，输出反馈则更接近实际测量条件。对同一个小型系统做多组增益比较，有助于区分可行性结论与闭环响应质量。

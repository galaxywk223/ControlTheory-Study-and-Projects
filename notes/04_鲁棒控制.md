# 鲁棒控制

鲁棒控制关注模型偏差、参数漂移和外部扰动存在时的闭环稳定性与性能保持能力。这篇笔记将几类常见性能指标统一到同一个分析框架下，并用区间不确定系统做数值验证。对应实验见 [`experiments/robust_control/04_robust_control`](../experiments/robust_control/04_robust_control/README.md)。

## 基本模型

考虑系统

```math
\dot x(t)=Ax(t)+Bu(t)+D\omega(t), \qquad
y(t)=Cx(t), \qquad
z(t)=Ex(t).
```

其中 $\omega(t)$ 表示外部扰动，$z(t)$ 表示性能输出。鲁棒控制的核心任务，是在不确定性和扰动同时存在时，仍然保证闭环满足稳定性和性能要求。

## 常见性能指标

### $\mathcal{H}_\infty$ 性能

若系统满足

```math
\int_0^t z^T(s)Sz(s)\,ds
\le
\gamma^2\int_0^t \omega^T(s)\omega(s)\,ds,
```

则可理解为输出能量被扰动能量所控制。$\gamma$ 越小，抗扰性能通常越强。

### $\mathcal{L}_2-\mathcal{L}_\infty$ 性能

若系统满足

```math
\sup_{t\ge 0} z^T(t)z(t)
\le
\gamma^2\int_0^t \omega^T(s)\omega(s)\,ds,
```

则说明在扰动能量有界时，输出峰值也受到限制。

### 无源与耗散

无源性从输入输出能量交换角度刻画系统；耗散理论则进一步用供给率统一描述不同性能。$\mathcal{H}_\infty$、无源性和更一般的 $(Q,S,R)$ 耗散性能，都可以通过合适的供给率落到同一类李雅普诺夫不等式框架下。

## 分析思路

这些结论的共同主线是：

1. 构造二次型李雅普诺夫函数 $V(t)=x^T(t)Px(t)$；
2. 把性能指标写成积分不等式；
3. 将 $\dot V$ 与性能函数合并，整理成关于 $\begin{bmatrix}x & \omega\end{bmatrix}^T$ 的二次型；
4. 通过 Schur 补得到可检验的矩阵不等式。

因此，很多鲁棒控制结果在形式上不同，本质上都依赖同一种“李雅普诺夫函数 + 性能供给率 + LMI”结构。

## 参数不确定性

鲁棒控制中常见的不确定性包括：

- 范数有界不确定性
- 区间不确定性
- 凸多面体不确定性

区间不确定性便于数值实验，因为可以直接在顶点和网格上检查稳定性与性能变化。若一个公共李雅普诺夫矩阵同时适用于多个顶点模型，往往就能得到整片参数区域的鲁棒结论。

## 数值结果

实验考虑区间不确定系统

```math
\dot x(t)=A(\rho_1,\rho_2)x(t)+Bu(t)+D\omega(t), \qquad z(t)=Ex(t),
```

其中

```math
A(\rho_1,\rho_2)=
\begin{bmatrix}
0 & 1 \\
1.2+\rho_1 & 0.3+\rho_2
\end{bmatrix},
\qquad
\rho_1\in[-0.6,0.6], \quad \rho_2\in[-0.25,0.25],
```

```math
B=D=
\begin{bmatrix}
0 \\
1
\end{bmatrix},
\qquad
E=
\begin{bmatrix}
1 & 0 \\
0 & 0.4
\end{bmatrix}.
```

采用状态反馈

```math
u(t)=Kx(t), \qquad
K=
\begin{bmatrix}
-3.9276 & -3.4536
\end{bmatrix}.
```

## 小结

鲁棒控制的核心不是针对某一个精确模型优化，而是为一类模型同时提供稳定性和性能保证。对区间不确定系统而言，公共李雅普诺夫函数、顶点判定和频域/时域联合验证，构成一条常用的分析路径。这组实验展示了同一个反馈增益如何在整片参数区域内稳定系统并抑制扰动影响。
